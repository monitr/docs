Integrations

Before you can use Monitr, you will have to set up an integration to connect to your accounting software.
To setup your integration, navigate to  Integrations in the sidebar.

Add an integration#

Click at the top of the page and select your accounting software provider.

  • Yuki
  • Exact Online
  • Afas (Soon)

You will need to create a Yuki API key to be able to add the integration. You can find more information on the documentation page of Yuki. Once you've created an API key, fill it in and click .

Once you've added your integration, we will automatically start importing all required data from your accounting provider in the background.



Data synchronisation#

Monitr will get data from your accounts, analytical accounts and general ledger each night. This way your dashboard will always give you an up to date view on your accounts. The data will be updated from the last actuals date that you can set in the  Entities settings. If data prior to the last actuals needs to be updated, change the last actuals to an earlier date, the data will refresh overnight. Set your last actuals to the correct date, after the data has been refreshed.

In addition, you can sync your data manually by clicking the button in the  Entities settings.
