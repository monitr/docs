Cash Flow

What is a Cash flow statement?#

A Cash flow statement gives an overview of the sources and the use of funds for a certain period of time. You can create a Cash flow statement in two ways:

  1. Direct Cash flow statement: allocating each transaction in your cash & bank to a category
  2. Indirect Cash flow statement: calculated based on the transactions in your general ledger accounts (other then your cash & bank)

Monitr uses the indirect method to calculate the Cash flow. We will allocate transactions from the Profit & loss statement and the Balance sheet to categories in the Cash flow statement. Cash & Bank accounts, and accounts that don't have a cash impact (see some examples here)

Setup a Cash flow statement#

Monitr allows you to tailor the Cash flow statement entirely to your needs. Go to tab 'Reporting' - 'Cash Flow'. This can either be done via a CSV upload or in the user interface (see below)

The Cash Flow table contains three fields:

  • Rank: the order in which lines of the cash flow are shown in dashboards
  • Name: the label that will be shown in dahboards for this line of the cash flow
  • Parent: This field is used to manage the subtotals of your Cash flow statement. e.g. to create a subtotal 'Total cash flow' of the lines 'Operational cash flow' and 'Financial cash flow'
    • Create a line ''Total cash flow'
    • Enter 'Total cash flow' in the field 'Partent' of the 'Operational cash flow' and 'Financial cash flow'
  • Via User Interface
  • Via CSV

You can either upload a pre-defined template (such as this file) or use tabsheet 'Cash flow' our google sheets template to define your Cash flow statement.

Definition#

Complete the folowing fields in the template

  • ID [string]: a unique identifier for the line in your cash flow statement
  • Name [string]: the label shown in the reporting
  • Rank [integer]: defines the order in which the items are shown
  • Parent [string]: used to define subtotals in the reporting. Enter the ID of the parent

Validation#

Make sure these validations rules are respected:

 ID needs to be unique
 Name is not empty
 Rank needs to be an integer and unique
 Parent any PartentID used should exist in ID
 The value of Parent cannot be the same as ID

To create a Cash flow statement via de Monitr user interface:

  1. Go to tab 'Reporting' - 'Cash Flow'.
  2. Add rows by clicking ' insert row'
    1. To insert a row at a specific place select the row above where you want to insert the row (a checkmark will be visible)
    2. Click ' insert row'
  3. Give the cash flow reporting line a name and if required link it to a Parent
  4. Repeat this untill you have made the Cash flow statement you need
  5. Click
  6. If any errors occur they will be highlighted in red. Your report will not be saved.
  7. Fix the errors and click untill the report is succelfully saved

To change the ranking of a line in a Cash flow statement you can drag and drop the line you want to move.
