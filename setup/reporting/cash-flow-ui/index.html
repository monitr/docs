cash-flow-ui

To create a Cash flow statement via de Monitr user interface:

  1. Go to tab 'Reporting' - 'Cash Flow'.
  2. Add rows by clicking ' insert row'
    1. To insert a row at a specific place select the row above where you want to insert the row (a checkmark will be visible)
    2. Click ' insert row'
  3. Give the cash flow reporting line a name and if required link it to a Parent
  4. Repeat this untill you have made the Cash flow statement you need
  5. Click
  6. If any errors occur they will be highlighted in red. Your report will not be saved.
  7. Fix the errors and click untill the report is succelfully saved

To change the ranking of a line in a Cash flow statement you can drag and drop the line you want to move.
