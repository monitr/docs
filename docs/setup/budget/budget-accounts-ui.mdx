To create a Budget Accounts via the Monitr user interface:
1. [Go to tab 'Budget Accounts'](https://app.monitr.be/dashboard/organisations/entities/definitions/budget/accounts).
1. Add rows by clicking '<Icon iconName="AddLink" /> insert row'
1. Give the Budget Accounts a name link it to a Mapping
1. Repeat this untill you have made the Budget Accounts you need
. Click <PrimaryButton>Save</PrimaryButton>
1. If any errors occur they will be highlighted in red. Your data will not be saved.
1. Fix the errors and click <PrimaryButton>Save</PrimaryButton> untill the report is succelfully saved


## Validation 
The following validation checks are performed:
<Icon iconName="SkypeCheck"/> The name of the Budget Account should be unique
