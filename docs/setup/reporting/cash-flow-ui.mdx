To create a Cash flow statement via de Monitr user interface:
1. [Go to tab 'Reporting' - 'Cash Flow'](https://app.monitr.be/dashboard/organisations/entities/definitions/mapping).
1. Add rows by clicking '<Icon iconName="AddLink" /> insert row'
    1. To insert a row at a specific place select the row above where you want to insert the row (a checkmark will be visible)
    1. Click '<Icon iconName="AddLink" /> insert row'
1. Give the cash flow reporting line a name and if required link it to a Parent
1. Repeat this untill you have made the Cash flow statement you need
1. Click <PrimaryButton>Save</PrimaryButton>
1. If any errors occur they will be highlighted in red. Your report will not be saved.
1. Fix the errors and click <PrimaryButton>Save</PrimaryButton> untill the report is succelfully saved

To change the ranking of a line in a Cash flow statement you can drag and drop the line you want to move.

## Validation 
The following validation checks are performed:
<Icon iconName="SkypeCheck"/> There should not be any circular references made via the 'Parent' fields
