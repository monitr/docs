## Setup your mapping
To create your mapping via the Monitr user interface:
1. [Go to tab 'Mapping'](https://app.monitr.be/dashboard/organisations/entities/definitions/mapping).
1. You can complete pre-filled rows highlighted in blue or add rows by clicking '<Icon iconName="AddLink" /> insert row'
1. Select the report line to which you want to allocate the mapping in the column 'Report line'. 
1. Click <PrimaryButton>Save</PrimaryButton>
1. If any errors occur they will be highlighted in red. Your report will not be saved.
1. Fix the errors and click <PrimaryButton>Save</PrimaryButton> untill the report is succelfully saved
